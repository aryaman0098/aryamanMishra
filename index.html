Recent Projects

Face-Mask Detection

This is a simple Face-mask recognition project which checks whether the person is wearing a face mask or not. The model is a Convolutional Neural Network which is used to classify whether the person is wearing a mask or not. The real-time testing is done with the aid of OpenCv library, which feeds the input from the web cam into the CNN model and predicts the output.

Neural Style Transfer

This project is an implementation of the paper A Neural Algorithm of Artistic Style. It uses Convolutional Neural Network to take the art form of one image (style image) and apply it another image (content image) The code is written in Python and makes use of the Tensorflow library.

Sudoku Game

This is a simple sudoku game made in C++ language which uses the backtracking algorithm to solve the sudoku game. The GUI for this game has been made using the QT framework. This game comprises of three levels of difficulty, which the player can choose according to his/her liking.
